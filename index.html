1.Write the code to access element which is having id as "text"

Ans.

        let element = document.getElementById("text");

Explanation:Here, we use the `getElementById` method provided by the `document` object to get the element with the id "text". The returned value is assigned to the `element` variable, which can then be used to manipulate the element as needed. In summary, the `getElementById` method allows us to access an element in the HTML document using its unique id attribute, and then manipulate it using JavaScript.

8.Whatâ€™s the difference between window vs document?

Ans.

In JavaScript, `window` and `document` are both important objects that are used to interact with the web page. `window` refers to the entire browser window that contains the document, including the address bar, tabs, and other UI elements. It is the top-level object in the browser's object model and provides global methods and properties that can be used to manipulate the window itself and its contents. On the other hand, `document` refers to the web page that is loaded in the browser window. It is a child object of the `window` object and provides methods and properties that can be used to manipulate the elements and content within the document, such as creating or modifying elements, handling events, or manipulating the document's structure. In summary, `window` is the top-level object that represents the entire browser window, while `document` is a child object of `window` that represents the web page loaded in the browser and provides methods to manipulate its contents.


9.Write code to implement timer clock using JS -- display HH:MM:SS -- the time should keep updating every second

Ans.

        function updateTime() {
            let today = new Date();
            let hours = today.getHours().toString().padStart(2, '0');
            let minutes = today.getMinutes().toString().padStart(2, '0');
            let seconds = today.getSeconds().toString().padStart(2, '0');
            let timeString = `${hours}:${minutes}:${seconds}`;
            document.getElementById('clock').textContent = timeString;
          }
          setInterval(updateTime, 1000);

Explanation:In this code, we define a function called `updateTime` that gets the current time using the `Date` object and formats it as a string in HH:MM:SS format. We then use the `getElementById` method to select an HTML element with an `id` of `clock` and update its `textContent` property to the current time string. Finally, we use the `setInterval` method to call the `updateTime` function every second (1000 milliseconds) to update the clock with the current time. In summary, we use the `Date` object to get the current time, format it as a string in HH:MM:SS format, and update an HTML element's `textContent` property with the current time string every second using `setInterval`.
